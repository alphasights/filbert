# Filbert

Filbert downloads a backup of a follower database for a given heroku application and stores it in `~/.heroku_backups`

## Installation

    gem install filbert

And reload your path (e.g. `rbenv rehash`). Then run


## Usage

    filbert backup --app your-heroku-appname
